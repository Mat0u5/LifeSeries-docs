## **Added Nice Life**:
### Snow Stuff
Snow layers accumulate infinitely.
Snow and ice now generates in all biomes at all times when the session is on.
Rain now renders as snow in all biomes.
Less obtrusive snowfall texture.
Snow now accumulates everywhere in layers, rather than randomly.
Snow can replace grass and flowers.

### Day-night
The day-night cycle now pauses at midnight.
You cannot skip the night until all players sleep at midnight.
You cannot get out of bed after all players go to sleep.

### Trivia
The santa bot spawns once all players go to sleep.
New trivia quiz screen.
If you answer a question correctly, you get the option to vote for who's on the nice / naughty list.


### Other
The nether is now also a winter (not-so)wonderland.
Removed nether fog.
Retextured nether gold ore and quartz.

### Configs:
'nice_life_less_snow' client side config. Default value is true.
'snow_when_not_in_session' config. Default value is false.
'snow_layer_increment_delay' config. Default value is 600 seconds.
'advance_time_not_in_session' config. Default value is false.
'snowy_nether' config. Default value is true.
